Requirements
------------
* Xlib header files
* ohsnap & terminus fonts

Instructions
------------

	git clone https://github.com/KieranQuinn/dwm.git
	cd dwm && sudo make clean install
	
	
What it includes
----------------
* statuscolors
* statusallmons
* centredfloating
* savefloats
* notitle
* pertag2
* systray
* occupiedcol
* uselessgaps
* keysymfix
* bstack
* runorraise
